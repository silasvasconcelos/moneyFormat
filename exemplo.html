Money Format

Formata moeda sem limit de numeros!
de 1000 => 1.000 ou 100000000000000 => 1.000.000.000.000,00
segundo os exeplos abaixo podese passar as formatações

Exemplo de uso

<script type="text/javascritp"> alert( moneyFormat(999999999) ); // 9.999.999,99 </script>

<script type="text/javascritp"> alert(moneyFormat(999999999,'.',',',2)); // 9.999.999,99 </script>

<script type="text/javascritp"> alert(moneyFormat(999999999,',','.',3)); // 999.999,999 </script>

Em input

<input type="text" onkeyup="this.value = moneyFormat(this.value);" placeholder="00,00" />
